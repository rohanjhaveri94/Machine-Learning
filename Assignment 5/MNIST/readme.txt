
A small subset of the MNIST digit classification dataset.  The data represent images that are 28x28 and the features were normalized to be between 0 and 1.

Files for training:
MNIST_train.csv :  features for training
MNIST_train_labels.csv : the labels (0 through 9) for the training data

Files for testing:
MNIST_test.csv : features for testing
MNIST_test_labels.csv : the labels (0 through 9) for the test data
